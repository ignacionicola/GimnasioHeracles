# API PARA GIMNASIO HERACLES
Api creada para la materia PROYECTO INTEGRADOR 2

## Funcionalidades
- Gestión de usuarios
    - Registro y Login de usuarios
    - Hash de contraseñas
    - Autenticación con JWT (roles: admin, cliente)
    - Middleware para proteger rutas y restringir acceso según su rol

- Gestión de socios
    - CRUD de productos para el admin
    - Validación de datos

## Tecnologías
- Javascript
- NodeJS
- Express
- Dotenv
- MySQL con sequelize
- JWT
- Cors
- Crypto
- Cookie-parser
* Dependencias de desarrollo:
    - Nodemon
